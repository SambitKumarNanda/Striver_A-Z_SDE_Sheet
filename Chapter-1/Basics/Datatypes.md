# Datatypes in Java

## <u>Java has 2 datatypes</u>:-

- Primitive Datatypes
- Non-Primitive Datatypes

### Primitive Datatypes
- It could be defined as the most basic types of datatype that is used while programming.
- It includes:-
  - int
  - char
  - float
  - double
  - long
  - byte
  - boolean

### Non-Primitive Datatypes
- These are the more complex kind of datatypes.
- It includes:-
  - Arrays
  - Strings 
  - Classes
